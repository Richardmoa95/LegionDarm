2.2 Contenido

Hemos concluido la estructura básica de nuestro sitio web, pero lo que realmente importa en una página web es lo que hay dentro de esas secciones, como títulos, textos, imágenes y vídeos, HTML define varios elementos para representarla.

Texto

HTML define varios elementos para determinar cada palabra, frase, o párrafo en el documento. El siguiente elemento se usa para representar títulos.

  • <h1>-- Representa un título. El título se declara entre las etiquetas de apertura y cierre. Incluye elementos adicionales para representar subtítulos, hasta seis niveles (<h2>, <h3>, <h4>, <h5>, y <h6>). 

                    
                        <header>
                        <h1>Este es el título</h1>       
                        </header>

Los siguientes son los elementos que ofrece HTML para representar el cuerpo del texto.

  • <p>-- Representa un párrafo.

  • <pre>-- Representa un texto con formato predefinido, como código de programación o un poema que requiere espacios asignados a cada carácter y saltos de línea.

  • <span>-- Puede contener un párrafo, una frase o una palabra. No aplica ningún estilo al texto pero se usa para asignar estilos personalizados 

                    
                        <aside>
                        <p>Cita del artículo uno</p>
                        <p>Cita del artículo dos</p>       
                        </aside>

Debido a que no todas las palabras en un texto tienen el mismo énfasis, HTML incluye los siguientes elementos para declarar un significado especial a palabras individuales o frases.

  • <em>-- Se usa para indicar énfasis. El texto se muestra con letra cursiva.

  • <strong>-- Es utilizado para indicar importancia. El texto se muestra en negrita.

  • <i>-- Representa un pensamiento, un término técnico, etc. El texto se muestra con letra cursiva.

  • <u>-- Representa texto no articulado. Se muestra subrayado.

  • <b>-- Se usa para indicar importancia. Debería ser implementado solo cuando ningún otro elemento es apropiado para la situación. El texto se muestra en negrita.

Estos elementos se pueden utilizar para resaltar títulos o etiquetas, o para destacar palabras o frases en un párrafo, según muestra el siguiente ejemplo. 

                    
                        <article>
                        <p>
                            La muerte es una <em>quimera</em>: porque mientras yo existo, no existe la <i>muerte</i>; 
                            <br>y cuando existe la <i>muerte</i>, <strong>ya no existo yo</strong>.<br>Epicuro de Samos
                        </p>
                        </article>

HTML incluye los siguientes elementos para insertar textos que tienen un propósito claramente definido.

  • <mark>- Resalta texto que es relevante (por ejemplo, términos que busca el usuario).

  • <small>- Representa letra pequeña, como declaraciones legales, descargos, etc.

  • <cite>- Representa el autor o título de una obra, como un libro, una película, etc.

  • <address>- Representa información de contacto. Se implementa con frecuencia dentro de los pies de página.

  • <time>- Representa una fecha, incluye el atributo datetime para especificar un valor en formato de ordenador y el atributo pubdate, el cual indica que el valor asignado al atributo datetime representa la fecha de publicación.

  • <code>- Representa código de programación. Se usa en conjunto con el elemento <pre>.

  • <data>- Representa datos genéricos. Puede incluir el atributo value para especificar el valor en formato de ordenador (por ejemplo, <data value="32">Treinta y Dos</data>). 

                    
                        <article>
                        <header>
                            <h1>Título del artículo</h1>
                            <time datetime="2016-10-12" pubdate>publicado 12-10-2016</time>
                        </header>
                        <p>
                            La muerte es una quimera: porque mientras yo <mark>existo</mark>, no existe la muerte; 
                            <br>y cuando existe la muerte, ya no existo yo.<br><cite>Epicuro de Samos</cite>
                        </p>
                    </article>

Enlaces

Como mencionamos anteriormente, un enlace es contenido asociado a una URL que indica la ubicación de un recurso. HTML incluye el siguiente elemento para crear enlaces.

  • <a>-- Crea un enlace. El texto o la imagen que representa el enlace se incluye entre las etiquetas de apertura y cierre. El elemento incluye el atributo href para especificar la URL del enlace.

Los enlaces pueden conectar el documento actual con otros. Por ejemplo, podemos enlazar las opciones en el menú de nuestra página web a otros documentos en nuestro servidor. 

                    
                        <nav>
                        <a href="index.html">Principal</a> |
                        <a href="index.html">Fotos</a> | 
                        <a href="videos.html">Videos</a> |       
                        <a href="contacto.html">Contacto</a>              
                        </nav>

El elemento <a> puede incluir el atributo target para especificar el destino en el cual el documento será abierto. El valor _self se asigna por defecto, significa que el documento se abre en la misma ubicación (el mismo recuadro o ventana). Otros valores son _blank (se abre en una nueva ventana), _parent (se abre en el recuadro padre), y _top (se abre en la ventana actual). Debido a que los recuadros no se usan de forma frecuente en sitios web modernos, los dos valores más comunes son _self, y _blank. El siguiente ejemplo implementa el último valor para acceder al dominio www.jdgauchat.com desde una nueva ventana. 

                    
                        <footer>
                        <address>Toronto, Canada</address>
                        <small>
                            &copy; 2016 
                            <a href="http://www.jdgauchat.com" target="_blank">J.DGauchat</a>
                        </small>
                        </footer>

Los documentos a los que se accede a través de un enlace creado por el elemento <a> son descargados por el navegador y mostrados en pantalla, pero a veces los usuarios no necesitan que el navegador abra el documento, sino que el archivo se almacene en sus discos duros para usarlo más adelante. HTML ofrece dos atributos para este propósito:

  • download-- Atributo booleano que indica que en lugar de leer el archivo el navegador debería descargarlo.

  • ping-- Declara la ruta del archivo que se debe abrir en el servidor cuando el usuario hace clic en el enlace. El valor puede ser una o más URL separadas por un espacio.

Cuando el atributo download se encuentra dentro de un elemento <a>, el archivo especificado por el atributo href se descarga y almacena en el disco duro del usuario. Por otro lado, el archivo que indica el atributo ping no se descarga, sino que se ejecuta en el servidor. Este archivo se puede usar para ejecutar código que almacena información en el servidor cada vez que el archivo principal se descarga o para llevar un control de las veces que esta acción ocurre. En el siguiente ejemplo, implementamos ambos atributos para permitir al usuario descargar un archivo PDF. 

                    
                        <article>
                        <p>
                            La muerte es una quimera: porque mientras yo existo, no existe la
                            muerte;<br>y cuando existe la muerte, ya no existo yo.<br>Epicuro de
                            Samos
                        </p>
                        <footer>
                            <a href="http://www.formasterminds.com/content/miarchivo.pdf"
                            ping="http://www.formasterminds.com/control.php" download>
                                Clic aquí para descargar
                            </a>
                        </footer>
                        </article>

Imágenes

Las imágenes se pueden considerar el segundo medio más importante en la Web. HTML incluye los siguientes elementos para introducir imágenes en nuestros documentos.

  • <img>-- Inserta una imagen en el documento. Requiere del atributo src para especificar la URL del archivo con la imagen que queremos incorporar.

  • <picture>-- Inserta una imagen en el documento. Trabaja junto con el elemento <source> para ofrecer múltiples imágenes en diferentes resoluciones. Útil para crear sitios web adaptables, como veremos en el Capítulo 5.

  • <figure>-- Representa contenido asociado con el contenido principal, pero que se puede eliminar sin que se vea afectado, como fotos, vídeos, etc.

  • <figcaption>-- Introduce un título para el elemento <figure>.

Para incluir una imagen en el documento, solo necesitamos declarar el elemento <img> y asignar la URL del archivo al atributo src. 

                    
                        <article>
                        <p>
                            La muerte es una quimera: porque mientras yo existo, no existe la muerte; <br>y cuando existe la muerte, 
                            ya no existo yo.<br>Epicuro de Samos
                        </p>
                        <img src="miimagen.jpg">         
                        </article>

La imagen se representa en su tamaño original, pero podemos definir un tamaño personalizado usando el resto de los atributos disponibles para el elemento <img>.

  • width-- Declara el ancho de la imagen.

  • height-- Declara la altura de la imagen.

  • alt-- Permite especificar una lista de imágenes de diferentes resoluciones que el navegador puede cargar para el mismo elemento.

  • sizes-- Especifica una lista de media queries (consulta de medios) junto con distintos tamaños de imágenes para que el navegador decida qué mostrar según la resolución de la pantalla. Estudiaremos media queries y diseño web adaptable en el Capítulo 5.

  • crossorigin-- Establece las credenciales para imágenes de origen cruzado (múltiples orígenes). Los valores posible son anonymous (sin credenciales) y usecredentials (requiere credenciales). Estudiaremos la tecnología CORS y cómo trabajar con imágenes procedentes de diferentes orígenes en el Capítulo 11.

Los atributos width y height determinan las dimensiones de la imagen, pero si declaramos ambos valores sin considerar la proporción original de la imagen, el navegador deberá estirar o achatar la imagen para adaptarla . Para reducir la imagen sin cambiar la proporción original, podemos especificar uno solo de los atributos y dejar que el navegador calcule el otro. 

                    
                        <article>
                        <p>
                            La muerte es una quimera: porque mientras yo existo, no existe la muerte; <br>y cuando existe la muerte, 
                            ya no existo yo.<br>Epicuro de Samos
                        </p>
                        <img src="miimagen.jpg" width="150" alt="Arboles en mi patio">         
                        </article>

Listados

A menudo, la información se debe representar como una lista de ítems. Por ejemplo, listados de libros, películas, o términos y descripciones. Para crear estos listados, HTML ofrece los siguientes elementos.

  • <ul>-- Crea una lista de ítems sin orden. Compuesto por etiquetas de apertura y cierre (<ul> y </ul>) y trabaja junto con el elemento <li>.

  • <ol>-- Crea una lista ordenada de ítems. Compuesto por etiquetas de apertura y cierre (<ol> y </ol>) y trabaja junto con el elemento <li>. Puede incluir los atributos reversed para invertir el orden, start para determinar el valor desde el cual los indicadores tienen que comenzar a contar y type para determinar el tipo de indicador que son 1 (números), a (letras minúsculas), A (letras mayúsculas), i (números romanos en minúsculas) e I (números romanos en mayúsculas).

  • <dl>-- Crea una lista de términos y descripciones. Trabaja junto con los elementos <dt> y <dd>. <dl> define la lista, <dt> define los términos y el elemento <dd> define las descripciones.

El elemento que usamos depende del contenido. Por ejemplo, si el orden no es importante, podemos usar el elemento <ul>. En esta clase de listas, los ítems se declaran entre las etiquetas <ul> con el elemento <li>, como el siguiente ejemplo. 

                    
                        <aside>
                        <ul>
                            <li>IT, Stephen King</li>   
                            <li>Carrie, Stephen King</li>   
                            <li>El Resplandor, Stephen King</li>   
                            <li>Misery, Stephen King</li>   
                        </ul>      
                        </aside>

Tablas

Se recomienda usar tablas para definir información tabular, como estadísticas o especificaciones técnicas. HTML incluye varios elementos para crear una tabla.

  • <table>-- Define una tabla. Incluye etiquetas de apertura y cierre para agrupar los elementos que definen la tabla.

  • <tr>-- Define una fila de celdas. Incluye etiquetas de apertura y cierre para agrupar las celdas.

  • <td>-- Define una celda. Incluye etiquetas de apertura y cierre para delimitar el contenido y puede incluir los atributos colspan y rowspan para indicar cuántas columnas y filas ocupa la celda.

  • <th>-- Define una celda para la cabecera de la tabla. Incluye etiquetas de apertura y cierre para delimitar el contenido y puede incluir los atributos colspan y rowspan para indicar cuántas columnas y filas ocupa la celda.

Para incluir una tabla primero tenemos que declarar el elemento <table> y luego los elementos <tr> y <td>, como muestra el siguiente ejemplo. 

                    
                        <article>

                                <td>Stephen King</td>

                            </tr>

                                <td>Carrie</td>

                                <td>El Resplandor</td>

Atributos Globales

La mayoría de los navegadores automáticamente traducen el contenido del documento cuando detectan que se ha escrito en un idioma diferente al del usuario, pero la página puede incluir frases o párrafos que no se deben alterar. Para controlar el proceso de traducción, HTML ofrece un atributo global llamado translate. Este atributo puede tomar dos valores: yes y no. Por defecto, el valor es yes (si). En el siguiente ejemplo, usamos un elemento <span> para especificar la parte del texto que no se debería traducir. 

                    
                        <p>My favorite movie is<span translate="no">Two Roads</span></p>

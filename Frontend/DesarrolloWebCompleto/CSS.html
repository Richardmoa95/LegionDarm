¿Qué hace un :root en CSS?

En CSS, :root es un pseudo-selector que hace referencia al elemento raíz del documento HTML. Es decir, :root se refiere al elemento <html> en el documento HTML. Se utiliza para establecer variables CSS globales que se pueden utilizar en todo el documento, y que se pueden actualizar y modificar fácilmente en un solo lugar. La sintaxis para definir una variable CSS es la siguiente: 

                
                    :root {
                        --variable: valor;
                      }

Por ejemplo, si queremos definir una variable para el color principal de nuestro sitio web, podemos hacer lo siguiente: 

                
                    :root {
                        --color-primario: #007bff;
                      }

Luego, podemos utilizar esta variable en cualquier lugar del documento: 

                
                    h1 {
                        color: var(--color-primario);
                      }

De esta manera, si queremos cambiar el color principal de nuestro sitio web, solo tenemos que modificar la definición de la variable en el selector :root, en lugar de tener que cambiar el valor en cada lugar donde se utiliza la variable. Es importante tener en cuenta que :root solo se utiliza una vez en el documento y debe estar en la parte superior del archivo CSS.
